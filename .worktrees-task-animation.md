# Task — High-quality animation system (easing + effects + transitions)

## Goal
Rewrite the animation system in src/nf-core/animation/ to be production-quality, inspired by GSAP's design. Must work with the new scene system and be verifiable frame-by-frame via recorder.

## Requirements

### Easing (src/nf-core/engine/keyframes.js — already exists, ENHANCE)
- Keep existing 10 easings (linear, easeIn, easeOut, easeInOut, spring, bounce, elastic, expo, back, circ)
- Add: springConfigurable(t, {damping, stiffness, mass}) — physics-based spring
- Add: cubicBezier(x1,y1,x2,y2) — returns easing function
- Add: steps(n) — discrete step easing for frame-by-frame
- ALL must be pure functions: f(t) → value, where t is 0-1
- Export both as object (EASINGS) and individually

### Effects (src/nf-core/animation/effects/ — already has 14, ENHANCE)
Each effect: apply(progress, opts) → {opacity, transform, filter, clipPath} CSS object
NOT Canvas ctx operations — must return CSS for DOM/SVG compositing.

Current effects return Canvas ctx mutations. REWRITE them to return CSS property objects:
```js
// OLD (Canvas): 
export function fadeIn(ctx, progress) { ctx.globalAlpha = progress; }

// NEW (CSS):
export function fadeIn(progress) { return { opacity: progress }; }
export function springIn(progress) { 
  const s = 1 - Math.exp(-6*progress) * Math.cos(progress*Math.PI*4) * (1-progress);
  return { opacity: Math.min(1, progress*2), transform: `scale(${0.5+s*0.5})` };
}
```

Rewrite ALL 14 effects to return CSS objects. Keep same names.
Update effects/index.js to export the new format.

### Transitions (src/nf-core/animation/transitions/ — already has 13, ENHANCE)
Each transition: apply(progress, opts) → {layerA: CSSObj, layerB: CSSObj}
Returns CSS for BOTH outgoing and incoming layers.

```js
// dissolve
export function dissolve(progress) {
  return {
    layerA: { opacity: 1 - progress },
    layerB: { opacity: progress }
  };
}

// wipeLeft
export function wipeLeft(progress) {
  return {
    layerA: { clipPath: `inset(0 0 0 ${progress*100}%)` },
    layerB: { clipPath: `inset(0 ${(1-progress)*100}% 0 0)` }
  };
}
```

Rewrite ALL 13 transitions to return CSS objects.

## Technical Constraints
- All functions must be PURE — same input → same output
- No requestAnimationFrame, setTimeout inside animation functions
- CSS-based output (not Canvas ctx) so it works with DOM/SVG scenes
- Frame-pure: recorder calls __onFrame({time:X}) → page renders that exact frame
- No external dependencies

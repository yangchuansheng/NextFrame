NextFrame Design System

3D Glass + Dark Aurora — desktop application. Min font 12px, min opacity 0.50.

1. Typography Scale
Desktop-first. Three font families. Minimum font size is 12px. Nothing goes below.
H1 — 28px Bold --fs-h1 · --t100
H2 — 20px Semibold --fs-h2 · --t100
H3 — 16px Semibold --fs-h3 · --t100
Body — 14px Regular. Default reading size for interfaces. --fs-body · --t80
Secondary — 13px. Descriptions, meta info, supporting text. --fs-sm · --t65
Minimum — 12px. Captions, badges, timestamps. Smallest allowed. --fs-xs · --t50
Serif reading — Noto Serif SC 18px, line-height 2.0. For scripts and long-form content. --fs-reading
Monospace — JetBrains Mono 13px. Data values, code, timestamps. --mono · --fs-sm
FORBIDDEN: font-size below 12px or text opacity below 0.50. Both are hard to read on dark backgrounds.

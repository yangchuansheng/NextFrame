# Review Instructions

Strict production reviewer for animation system rewrite.

## Steps
1. Read all rewritten effect files in src/nf-core/animation/effects/ — each must return CSS object, not Canvas ctx
2. Read all rewritten transition files in src/nf-core/animation/transitions/ — each must return {layerA, layerB} CSS objects
3. Read src/nf-core/animation/apply.js — must export getEffectCSS, getTransitionCSS, ease
4. Read src/nf-core/engine/keyframes.js — must have springConfigurable, cubicBezier, steps
5. Check output/test-animations.html exists and has __onFrame protocol
6. Run: ./target/release/nextframe-recorder slide output/test-animations.html --out output/test-animations.mp4 --width 1920 --height 1080 --dpr 1 --fps 30 --parallel 4
7. Verify recording succeeded
8. Extract frames at t=2,5,10,17 and verify content

## Scoring
10/10: All effects return CSS, all transitions return CSS, recording works, frame extraction shows correct visuals
< 10: Any Canvas ctx calls remain, recording fails, or effects don't produce visible output
